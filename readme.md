# TravelPlanMap/front

## Setup
* Create `.env` and Write Google Map API key
```
MAP_API_KEY=xxxxx
```

* Install node modules
```
npm install
```

## Scripts
### Build js and css
* Build for development mode
```
npm run dev
```
* Build for production mode
```
npm run build
```

### Check/Fix code format (*.js and *.vue)
* Check code format
```
npm run lint
```
* Fix code format
```
npm run format
```
